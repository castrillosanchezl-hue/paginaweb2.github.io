¿De qué trata nuestro proyecto?

Nuestro proyecto consiste en crear una página web y un juego interactivo, con el objetivo de ayudar a aprender y entender mejor la suma de fracciones.

Queremos que el aprendizaje sea más dinámico y divertido, usando herramientas digitales que permitan practicar y reforzar este tema de forma fácil y entretenida.

Con esta propuesta buscamos incentivar el interés por las matemáticas y facilitar el aprendizaje para todos los estudiantes y/o personas que simplemente quieren una forma diferente de aprender o reforzar sus conocimientos.

1. ¿Qué es una fracción?

Una fracción representa una parte de un todo. Se escribe como:

a / b

  • a es el numerador (las partes que tomas).
  • b es el denominador (en cuántas partes se dividió el todo).

2. Suma de fracciones con el mismo denominador

Cuando las fracciones tienen el mismo denominador, simplemente sumamos los numeradores y dejamos el mismo denominador.

a/b + c/b = (a + c) / b

Ejemplo:

2/7 + 3/7 = (2 + 3) / 7 = 5/7

3. Suma de fracciones con distinto denominador

Cuando los denominadores son diferentes seguimos estos pasos:

  1. Encontrar un denominador común (generalmente el producto o el MCM).
  2. Convertir cada fracción a una fracción equivalente con ese denominador común.
  3. Sumar los numeradores.
  4. Simplificar (si es posible) el resultado.

Ejemplo:

1/4 + 1/6

  • Denominador común: 12
  • 1/4 = 3/12
  • 1/6 = 2/12

1/4 + 1/6 = 3/12 + 2/12 = 5/12

4. Ejemplos resueltos

Ejemplo 1

3/5 + 1/5 = (3 + 1)/5 = 4/5

Ejemplo 2

2/3 + 1/6

Denominador común: 6
2/3 = 4/6 y 1/6 = 1/6
Entonces: 4/6 + 1/6 = 5/6
